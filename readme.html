CirQのご購読ありがとうございます。　
CirQは創刊に際して次のような方針を立てました。

　最近の電子技術、通信技術の進歩は目を見張るものがあります。その進歩が新しくそれらの技術を知り、楽しもうと考える人にとって高い障壁になっていることも事実です。
　一方、出版界の不景気もあって電子関係の入門雑誌が軒並み廃刊してしまい参考書の入手も難しいものになっております。　「工業立国日本」を標語としているわが国にとって「このままでは日本の将来はどうなるのだろう?」と心配する方々も多く見受けられます。
　これらの諸現象は社会の大きな流れにそったものであって「こうすれば改善される」と簡単に解決出来る問題ではありません。　しかし、それだからといって「そのまま大きな流れに流されるしか無い」と考えるのも面白くありません。少しでもそれらの技術を知り、楽しもうと考えている人たちの前にある障壁を取り崩すことができれば、と考えてこのCirQの発行を思い付きました。
　このCirQの編集方針は次の通りです。
　(1)　初心者にもなるべく分かりやすい言葉で記事を書く。
　(2)　実際に製作出来る簡単な回路を記事にする
　(3)　主題は電子、通信に関する事ですが、電子、通信のみでなく科学全般に目を向ける。
　(4)　「エッ?」と思うような話題を盛り込む。
　(5)　何より楽しい記事にしたい。
　(6)　新しい筆者の開拓もしたい。(投稿歓迎、ただし原稿料無料)
等々です。

　発行は当面、隔月とします。
　購読料は無料です。
